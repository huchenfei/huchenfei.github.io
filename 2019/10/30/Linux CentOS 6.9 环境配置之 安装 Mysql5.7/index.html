=====================卸载=====================
我们经常会遇到安装失败的结果,所以需要卸载.
1.先关闭Mysql

2.查看有哪些安装包组件

3.卸载这些组件

4.卸载组件完成之后, 还会有残存的注册表

5.删除这些注册信息

6.删除用户及用户组

7.检查是否卸载完全

==================================mysql的用户权限==================================
1.创建用户(通过 root 超级用户创建)

USERNAME 是 你创建的用户名, HOST 是 指定的访问环境, localhost 和 % 一个代表只能本地登录一个表示任意远程地址登录, PASSWORD 顾名思义表示 登录密码

这就说明创建了一个登录名 hello, 密码world, 可以任意远程地址登录的用户

ALL : 代表所有的权限赋予
DATABASENAME : 数据库名
TABLENAME : 表名
USERNAME : 用户名
HOST : 主机名 (localhost %)

Linux 配置环境之安装 JDK

发表于 2019-10-30 | 分类于 Linux
  • 1.检测是否安装了 JDK

打印出来下面那句话就是说明没有
-bash: jave: command not found

–

打印出来类似下面那句话就说明有 java version “1.8.0_121” Java(TM) SE Runtime
Environment (build 1.8.0_121-b13) Java HotSpot(TM) 64-Bit Server VM
(build 25.121-b13, mixed mode)

如果有预安装的JDK 进行步骤 2 , 没有进行步骤 4

  • 3.卸载预安装的 JDK

    • 3.1. 先卸载 openjdk 1.7

  • 4.安装 JDK

    • 上传 jdk的tar.gz包到服务器上,可以使用ftp上传, 也可以使用linux自带的rz 命令上传

      • 上传完成之后, 将gz包解压, 在这里先要将自己准备解压到的包mkdir 出来.

    • 因为我的gz包都放在了一个 installpackage 的目录下面, 所以我需要通过上面的方式来找到正确的位置, 如果你的包就在你要解压的目录下面, 就直接 # tar zxf jdk-8u121-linux-x64.tar.gz

    • 安装需要的依赖
    • # yum install glibc.i686
